GESTÃO EMPRESA - FASE 1

O que esta versão faz:
- Registar trabalhos
- Editar trabalhos
- Apagar trabalhos
- Ver dashboard com totais
- Pesquisar e filtrar trabalhos
- Exportar dados em JSON

Como usar no PC:
1. Extrair o ZIP
2. Abrir o ficheiro index.html no browser

Como usar no Android:
1. Enviar a pasta para um alojamento simples, GitHub Pages ou Netlify
2. Abrir no Chrome
3. Escolher "Adicionar ao ecrã principal"

Notas:
- Nesta fase os dados ficam guardados localmente no dispositivo
- Se quiseres, no próximo passo posso fazer uma versão com Firebase para sincronizar PC + Android
- Também posso transformar esta base em APK Android e versão desktop com Electron
