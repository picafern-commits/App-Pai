GESTÃO EMPRESA PRO - VERSÃO 2

O que esta versão inclui:
- Dashboard com totais
- Registo de trabalhos
- Base de clientes
- Registo de pagamentos
- Relatórios mensais
- Exportação CSV
- Backup JSON
- Estrutura pronta para Firebase
- Layout responsivo para PC e Android

COMO USAR:
1. Abrir o ficheiro index.html no navegador
2. Para melhor experiência no Android, abrir no Chrome e usar "Adicionar ao ecrã principal"
3. No PC, podes abrir normalmente ou alojar num GitHub Pages / servidor local

DADOS:
- Nesta versão os dados ficam guardados no navegador (localStorage)
- Podes exportar um backup JSON em qualquer altura
- Podes importar um backup depois

FIREBASE:
- Edita o ficheiro js/firebase-config.js
- Cola lá a configuração do teu projeto Firebase
- Esta base já está pronta para evoluir para sincronização cloud

PRÓXIMA FASE RECOMENDADA:
- Login por PIN
- Faturas / orçamentos PDF
- Fotos por trabalho
- Sincronização Firebase completa
- Multiutilizador
